# Usage
## via NPM
```bash
npm install randimg
```
## via CDN
```javascript
<script type="module">
    import HashMapsApi from "https://cdn.jsdelivr.net/gh/readyyyk/randImgAPI-npm@main/HashMapsApi.js";
    const hashMapsApi = new HashMapsApi("https://readyyyk-randimg.fly.dev");
    console.log(hashMapsApi.getLink("picsum"));
</script>
```
